# Included demo project

`night-storm-demo.dreamcue` is a self-contained sample generated entirely by this repository:

- the 18-second mono WAV is synthesized from sine waves;
- the five SVG images use programmatic gradients, circles, paths and original scene labels;
- no personal, downloaded or third-party media is included.

Rebuild it with:

```powershell
npm run build:demo
```

The sample exists to demonstrate the software. Its inclusion does not create a license for the rest of the repository. See the root README for the current licensing status.
